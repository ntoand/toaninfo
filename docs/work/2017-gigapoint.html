Gigapoint - display large point datasets

Platforms: Linux, MacOS

Built with: CMake, C++, Omegalib, OpenGL

Source: https://github.com/mivp/gigapoint

Gigapoint is an Omegalib module to display large points datasets (e.g. LIDAR data) on desktop and visualisation cluster e.g. the CAVE2.

Features

  • Use potree data as input
  • Asynchronous and multi-threading file loading
  • Support stereo display
  • Eye Doom Lighting (EDL) filter
  • Support live data update from a SLAM system
  • Support data interaction with multiple wands e.g. track finding
