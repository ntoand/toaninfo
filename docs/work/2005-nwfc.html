NWFC game engine and dead rising game

Platforms: Windows

Built with:

  • Microsoft VC++.NET 2002, DirectX 9.0c SDK Update Summer 2004
  • MS Windows 2000/XP (SP 1, 2)
  • Debugger library for debugging
  • 3D designer: 3DS Max 7.0, Maya 5.0, Deep Exploration 3.5
  • Physics engine: NovodeX SDK 2.2
  • Audio library GAudio

This is our thesis topic to fulfill the requirements of bachelor degree of Natural Sciences Univ. of Ho Chi Minh City. In this thesis, we did research on 3D game engine. We, then, developed and 3D game engine and a game demo in VC7 and DirectX 9.0c.
