Pair correlation

Platforms: Windows, MacOS, Linux

Built with: Matlab

This is an easy-to-use Matlab GUI program that can calculate auto and cross correlation1 similarly to “Carpet analysis” function in SimFCS program. The program can take fluorescence correlation spectroscopy data (.tiff) as input.
